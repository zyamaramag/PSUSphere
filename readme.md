# PSUSphere

PSUSphere is a Django-based web application for managing student organizations.

## Features
- College and program management
- Student and organization records
- Django admin dashboard with filters and search
- Fake data generation using Faker

## Technologies Used
- Python
- Django
- SQLite
- Faker
